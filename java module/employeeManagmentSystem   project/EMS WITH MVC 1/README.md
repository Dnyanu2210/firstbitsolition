# Employee Management System (EMS)

## Java Console Application using MVC-1 Architecture

---

## Project Overview

The **Employee Management System (EMS)** is a **console-based Java application** developed using the **MVC-1 architecture**.  
The project is designed to manage employee records such as **HR, Admin, and Sales Manager** using **Object-Oriented Programming concepts**.

Employee data is stored **in memory using arrays** and handled through a **DAO (Data Access Object) layer**.
